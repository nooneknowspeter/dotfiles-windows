## installation

### glazewm && zebar dir
```powershell
$HOME\.glzr
```
### alacritty
```powershell
$env:APPDATA\alacritty
```

### nvim
```powershell
$env:LOCALAPPDATA\nvim
```

### wezterm
```powershell
$HOME\.config\wezterm
```
